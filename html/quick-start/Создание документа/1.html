Для более удобной работы можно перейти в режим полного экрана, нажав на клавишу F11. Для выхода из полноэкранного режима нужно нажать F11 повторно.

Создадим новый документ. На боковой панели при запуске IDE будет открыт раздел Документ. Если его нет, нажмите на значок документа в левом верхнем углу. Нажмите «Создать...». Откроется диалоговое окно создания документа.
