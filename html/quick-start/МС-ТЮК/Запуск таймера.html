Добавьте запуск таймера. Чтобы у нас появилось событие «прошло 0,5 секунды», нужно запустить отсчет таймера. Для этого откройте редактор состояния Включен, зайдите внутрь уже созданного события и добавьте действие для таймера аналогичным образом. Это будет Timer.start. Здесь важно указать интервал срабатывания таймера, введите соответствующее число (в миллисекундах) в текстовое поле параметра. Кнопка слева от названия позволяет переключиться в другой режим выбора значения (что сейчас не требуется для задачи).

После того как добавили необходимые действия нажмите кнопку Сохранить.

Добавьте отключение светодиода в другом состоянии. Данное действие повторяет шаги выше и предлагаем вам самостоятельно пройти по пройденным шагам.

Останется лишь добавить событие, но перед этим приведем краткую инструкцию по универсальным действиям.

  • Чтобы отредактировать действие, дважды нажмите на него левой кнопкой мыши в списке в разделе Делай или непосредственно на графическом поле редактора.
  • Чтобы поменять действия местами, можно перетащить их в списке в разделе Делай.
  • Чтобы удалить действие в разделе Делай, выберите его и нажмите на кнопку с минусом.
  • Удаление по ошибке можно отменить, выйдя из редактора. На графическом поле удалить действие можно, нажав на нём правой кнопкой и выбрав Удалить. Отменить удаление можно нажатием сочетания клавиш Ctrl + Z.
