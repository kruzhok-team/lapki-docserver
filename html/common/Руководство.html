Советы по взаимодействию с руководством:

  • Нажмите кнопку Содержание, чтобы вернуться к меню выбора страниц руководства;
  • Нажмите кнопку Просмотр, чтобы вернуться из раздела Содержание к последней открытой странице;
  • Используйте кнопки Назад и Вперёд для последовательного перемещения по руководству;
  • Нажмите кнопку F1 на вашей клавиатуре для того, чтобы открыть, либо закрыть руководство;
  • Нажмите на кнопку руководства пользователя на боковой панели, чтобы закрыть, либо открыть руководство;
  • При наведении на левый край области руководства у Вас изменится указатель мыши, после чего, Вы можете зажать левую кнопку мыши и сдвинуть её влево (вправо), чтобы растянуть (сжать) область руководства по Вашему усмотрению.
  • Нажмите на кнопку Компоненты, чтобы открыть справочник по компонентам платформ.
