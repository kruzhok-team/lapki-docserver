03. Документация

В области документации находится два раздела:

  • Содержание — включает в себя дерево документации в котором при нажатии на какой-либо документ, справа  появится раздел «Просмотр»;
  • Просмотр — раздел для просмотра открытого документа и позволяющий переключаться между ними c помощью клавиш «Назад» и «Вперёд».

Чтобы открыть (закрыть) область документации нажмите на иконку вопроса.

При наведении на левый край области документации у Вас изменится указатель мыши, после чего, Вы можете зажать левую кнопку мыши и сдвинуть её влево (вправо), чтобы растянуть (сжать) область документации по Вашему усмотрению.

Также открыть (закрыть) область документации можно при нажатии клавиши «F1» на клавиатуре.
