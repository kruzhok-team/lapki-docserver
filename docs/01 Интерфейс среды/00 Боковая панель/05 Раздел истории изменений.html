05. Раздел истории изменений

После того как Вы создадите/откроете схему, в разделе История изменений отобразится две области:

  • Область Сделано содержит информацию о произведенный Вами изменениях с момента начала работы;
  • Область Отменено содержит информацию о последних отменённых Вами действиях;

Кнопка Назад отменяет Ваше последнее действие.

Кнопка Вперёд возвращает последнее отменённое Вами действие.

Для быстрой отмены последнего действия, нажмите сочетание клавиш Ctrl+Z.
Для быстрого возврата последнего отменённого действия, нажмите сочетание клавиш Ctrl+Y.
