03. Раздел машин состояний

Эта вкладка содержит кнопку Добавить и список машин состояний.

При нажатии на кнопку Добавить откроется окно добавления машины состояния, где можно ввести её имя и платформу.

Машина состояний не обязательно должна иметь имя, если его нет, то будет использоваться ID машины состояний.

Ниже кнопки Добавить находится список всех машин состояний. При нажатии правой кнопкой мыши на одну из машин состояний появится вкладка с её схемой. При двойном клике мышью на машине состояний открывается её окно редактирования, где можно узнать её платформу, изменить название, а также удалить её.

Удаление машины состояний нельзя отменить!
