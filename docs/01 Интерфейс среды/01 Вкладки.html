01. Интерфейс среды

01. Вкладки

После создания (открытия) схемы над основным полем редактирования появится область вкладок. Данная область предназначена для показа вкладок и переключения между ними, таких как:

  • 1. Вкладка редактора машины состояний;
  • 2. Вкладка просмотра кода машины состояний;
  • 3. Вкладка просмотра кода состояний (связей) находящиеся непосредственно в самой машине состояний.

Для того чтобы закрыть вкладку, достаточно нажать на крестик справа от наименования.

Для удобства есть перенос вкладок между собой, достаточно зажать левой кнопкой мыши любую из вкладок и перенести её поверх другой, после чего они поменяются местами, что позволит упорядочить приоритетность вкладок по своему усмотрению.
