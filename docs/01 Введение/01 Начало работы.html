00. Введение

01. Начало работы

Для начала рассмотрим каждый элемент интерфейса IDE.

В центральной области IDE до создания (открытия) схемы расположены подсказки, такие как сочетания клавиш. После открытия схемы появится поле редактирования машины состояний.

На левой панели сверху вниз расположены разделы:

  1. Документ — включает в себя создание схем, открытие файлов, импорт схемы из игры «Берлога: Защита пасеки», сохранение текущего проекта в файл, а также готовые примеры для тренировки;
  2. Проводник — здесь происходят взаимодействия с компонентами и иерархией состояний для удобства работы с элементами схемы в редакторе;
  3. Машины состояний — список машин состояний;
  4. Компилятор — преобразование схемы машины состояний в формат выбранной целевой платформы;
  5. Загрузчик — загрузка скомпилированной схемы машины состояний или стороннего файла в подключенное устройство;
  6. История изменений — возврат или отмена последних действий в редакторе IDE, а также их краткое описание;
  7. Настройки — содержит настраиваемые элементы IDE, например, тема приложения (светлая и тёмная), выбор адреса для компилятора и т.д.

После открытия или создания схемы на верхней панели появятся вкладка с машиной состояния. По ходу работы на этой панели могут появиться другие вкладки, такие как вкладки с кодом.

Правая панель открывается с помощью клика на знак вопроса в правом нижнем углу экрана, который при нажатии открывает справку по работе с IDE.
