04. Работа с загрузчиком

05. Загрузка прошивки в плату

Для прошивки нужен соответствующий файл. Его можно создать посредством компиляции схемы внутри IDE, а также взять снаружи (в случае с Arduino файл будет иметь расширение .hex). В первом случае вам достаточно перейти на вкладку загрузчика после компиляции, для прошивки внешнего файла нужно выбрать его с помощью кнопки .... Подключите к компьютеру прошиваемую плату или подключитесь к удалённому загрузчику. Выберите нужное устройство в списке и нажмите кнопку Загрузить.

Если вы открыли не тот файл, который Вам нужен для прошивки, то нажмите на кнопку X и выберите другую подходящую прошивку.

Если под кнопкой Загрузить, появилась надпись вида «из файла file.hex», где file.hex – это название файла, значит, этот файл будет выбран для загрузки в устройство. Если данная надпись отсутствует, для прошивки будет использоваться результат последней успешной компиляции схемы через IDE.

Начнётся загрузка прошивки, после её завершения плата Arduino начнёт выполнять программу, соответствующей загруженной схеме, а в левом нижнем углу раздела Загрузчик появится уведомление Загрузка завершена!, если прошивка не удалась, то в том же месте появится уведомление об ошибке.

Помните, что при работе с удалённым загрузчиком нельзя выбрать устройство, которое в данный момент прошивается другим клиентом. Если вы выбрали такое устройство, то загрузчик отклонит Ваш запрос.
