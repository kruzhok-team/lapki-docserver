02. Удаление компонента со cхемы

На левой панели перейдите на вкладку Проводник (значок в виде кубика), в появившейся вкладке выберите компонент, откройте модальное окно редактирования. В открывшемся окне нажмите Удалить. Выбранный компонент удалится со схемы, связанные с ним события будут обозначены особым значком.

Для быстрого удаление, нажмите левой кнопкой мыши на компонент в списке, затем нажмите комбинацию клавиш Ctrl+Del. Быстрое удаление работает так же и на Num Pad (обычно располагается с правой части клавиатуры).
