00. Добавление нового компонента

Нажмите на Проводник на боковой панели, в появившейся вкладке нажмите кнопку Добавить... Появится всплывающее окно выбора компонента.

Выберите компонент из списка. В случае, если компонент не является синглтоном (т.е. единственным для схемы), введите название компонента и нажмите Добавить. В списке компонентов появится новый компонент с соответствующим (в т.ч. введённым ранее названием) и соответствующим значком.
