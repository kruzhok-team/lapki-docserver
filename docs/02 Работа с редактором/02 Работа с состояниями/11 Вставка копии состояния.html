11. Вставка копии состояния

Нажмите правой кнопкой мыши на поле редактирования, в появившемся контекстном меню выберите «Вставить». В поле редактирования появится копия состояния.

Для быстрой вставки копии, нажмите левой кнопкой мыши на поле редактирования, затем нажмите сочетание клавиш «Ctrl+V».
