01. Открытие файла

Нажмите на «Документ» на боковой панели, в открывшейся вкладке выберите «Открыть». Откроется окно приложения для выбора файла. При выборе файла на поле редактирования появится схема, соответствующая записанной в файл.

Для быстрого открытия файла, нажмите сочетание клавиш «Ctrl+O».
