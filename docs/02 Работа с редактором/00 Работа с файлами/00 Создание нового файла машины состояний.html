02. Работа с редактором → 00. Работа с файлами

00. Создание нового файла машины состояний

Нажмите на Документ на боковой панели, в открывшейся вкладке выберите Создать, после выберите целевую платформу. На верхней панели появится вкладка редактора, также появится пустое поле редактирования схемы.

Для быстрого создания файла, нажмите сочетание клавиш Ctrl+N.
