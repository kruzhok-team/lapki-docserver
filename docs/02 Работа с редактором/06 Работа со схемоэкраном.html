02. Работа с редактором

06. Работа со схемоэкраном

Схемотехнический экран – это редактор модульной структуры системы устройств, которые прошивается из документа. По аналогии с игровыми движками, это «сцена», на которой размещены устройства с компонентами внутри.

На схемотехнический экране Вы можете увидеть, имеющиеся машины состояния с их компонентами.

Для того, чтобы открыть схемотехнический экран перейдите на вкладку Документ, затем нажмите на кнопку Открыть схемоэкран.
