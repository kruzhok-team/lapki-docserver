06. Работа с менеджером МС-ТЮК

07. Метаданные

Метаданные - это дополнительная информация об устройстве.

Для того, чтобы выполнить получить метаданные МС-ТЮК, нажмите на кнопку Получить метаданные. При получении ошибки, в текстовом поле появится сообщение с этой ошибкой, иначе будет получено сообщение с метаданными, они также будут добавлены в адресную книгу. Для того, чтобы их посмотреть нужно открыть адресную книгу и дважды кликнуть по устройству, метаданные которых были получены.

Перед выполнением команды, необходимо установить адрес устройства.

Метаданные содержат информацию о типе устройства, после их получения, в адресной книге в соответствующем столбце обновится информация о типе.
