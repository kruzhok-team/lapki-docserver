06. Работа с менеджером МС-ТЮК

04. Адресная книга

Адресная книга позволяет сохранить, загрузить и просмотреть дополнительную информацию об МС-ТЮК.

Для того, чтобы открыть адресную книгу нажмите на соответствующую кнопку, откроется таблица, которая содержит три столбца:

  1. Название: наименование устройства, которое задаётся пользователем
  2. Адрес: идентификатор устройства
  3. Тип: вид устройства

При двойном клике на какую-либо запись открывается его окно редактирования, где можно поменять название и адрес. Также можно посмотреть метаданные данного МС-ТЮК, при условии, что они известны.

Для того, чтобы добавить новую запись нужно нажать на кнопку +. Откроется окно добавления записи, в котором можно задать имя и адрес МС-ТЮК. После заполнения полей, нажмите на кнопку Добавить, чтобы сохранить запись. После этого её можно будет загрузить при следующем запуске IDE.

Название устройства - это не обязательное для заполнения поле, оно служит для удобства, чтобы было легче найти устройство.

Для того, чтобы удалить запись кликните на одну из записей в книге и затем нажмите на кнопку -.

Удаление записи нельзя отменить!

Для того, чтобы установить адрес для выполнения команд кликните на одну из записей и затем нажмите на кнопку Выбрать.

Выбирайте соответствующий устройству адрес, ведь иначе выполнить команды не получится.
