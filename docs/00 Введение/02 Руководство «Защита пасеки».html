00. Введение

02. Руководство «Берлога: Защита пасеки»

Научимся пошагово работать со схемой → с платформой «Берлога: Защита пасеки» в IDE.

Для лучшей работы в IDE перейдём сначала в полноэкранный режим, для этого нажмите на клавишу F11 на клавиатуре. Для выхода из полноэкранного режима нужно нажать F11 повторно.

  1. На боковой панели выберите раздел Документ (значок в виде трёх полосок);
  2. В появившейся вкладке нажимаем на Создать...;
  3. В диалоговом окне на первой вкладке выбираем схему для платформы «Берлога: Защита пасеки» - Автобортник. После выбора схемы, появится поле редактирования машин состояний;
  4. На боковой панели выберите раздел Проводник (значок в виде кубика);
  5. В появившейся вкладке нажмите кнопку Добавить... Появится диалоговое окно выбора компонента;
  6. Выберите компонент Сенсор и нажмите Добавить. В списке компонентов появится новый компонент;
  7. Проделываем то же действие из пункта №6 c компонентами АнализаторЦели, МодульДвижения, ОружиеЦелевое;
  8. Дважды нажимаем левой кнопкой мыши на поле редактирования. На месте клика появится новое пустое состояние;

    При создании первого состояния рядом с выбранным состоянием появится элемент Синего цвета, обозначающая его начальным процессом машины состояний. В любой момент по нажатию на правую кнопку мыши на любое из состояний, вы можете изменить начальное состояние (Редактировать→Назначить начальным).

  9. Добавьте ещё три состояния;

    Для быстрого создания подобного состояния, выберите уже созданое состояние и нажмите сочетание клавиш Ctrl+C, а затем Ctrl+V. Поверх копируемого состояния появится его копия, нажмите на неё один раз и перенесите её в сторону, чтобы оба состояния были видны в редакторе.

  10. Дважды нажимаем левой кнопкой мыши на заголовок любого состояния или один раз на иконку карандаша. После переименуйте состояние на Бой и нажимите на клавишу Enter, либо нажмите левой кнопкой мыши за пределами заголовка состояния;

    Для отмены переименования нажмите клавишу Esc.

  11. Переименуйте остальные состояния на Скан, Сближение, Атака;
  12. Назначьте Скан начальным состоянием. Для этого, нажмите правой кнопкой мыши на состояние Скан и выберите в появившемся меню опцию Редактировать→Назначить начальным;
  13. Под заголовком состояния Скан, дважды нажимаем левой кнопкой мыши на пустую область состояния. Появится диалоговое окно для выбора нового события и его действий;
  14. Вверху в левом выпадающем списке выберите событие System, а в правом выпадающем списке действие onEnter.
  15. Теперь нажмите кнопку с плюсом справа от области Делай. В диалоговом окне выберите действие Сенсор → ПоискВрагаПоДистанции, а в появившемся ниже блоке Направление поиска выберите мин. Нажмите Сохранить;

    В списке раздела Делай появится пиктографические изображения созданного Вами действия.

  16. Чтобы удалить действие из списка в разделе Делай, достаточно выбрать его и следом нажать кнопку с иконкой минуса. После чего, выбранное действие будет удалено из списка.
  17. Теперь самостоятельно создайте только что удалённое действие из пункта №14 и нажмите на кнопку Сохранить, после чего диалоговое окно закроется, а в состоянии появится Ваше событие;

    Если вы допустили ошибку в создании действия, то в списке при двойном нажатии на действие Вам откроется окно для его изменения.

  18. Дважды нажмите на состояние с уже созданным событием, в диалоговом окне создайте ещё одно событие System → onExit и добавьте действие Сенсор → ОстановкаПоиска;
  19. Нажмите дважды на состояние с наименованием Сближение, в диалоговом окне создайте событие System → onEnter и добавьте действие МодульДвижения → ДвигатьсяКЦели;
  20. Нажмите дважды на состояние с наименованием Атака, в диалоговом окне создайте событие System → onEnter и добавьте действие ОружиеЦелевое → АтаковатьЦель;
  21. Зажмите левой кнопкой мыши состояние с наименованием Сближение, перенесите его поверх состояния Бой и продержите так две секунды, затем отожмите мышку. Состояние сперва выделится синим цветом, затем после отпускания мышки состояние с наименованием Сближение окажется внутри состояния Бой;
  22. Зажмите левой кнопкой мыши состояние с наименованием Сближение, перенесите его поверх состояния Бой и продержите так две секунды, затем отожмите мышку. Состояние сперва выделится синим цветом, затем после отпускания мышки через две секунды состояние с наименованием Сближение окажется внутри состояния Бой;

    Чтобы вытащить состояния из другого достаточно зажать левой кнопкой мыши внутреннее состояние и слегка отвести в сторону. После чего продержать мышку две секунды и состояние тут же окажется независимым.

  23. Переместите состояние Атака внутрь состояния Бой;
  24. Наведитесь курсором на состояние с наименованием Атака и нажмите на него один раз, затем переведите курсор на появившуюся на рамке состояния иконку со знаком плюс и потяните до состояния с названием Сближение. Появится диалоговое окно для редактирования нового перехода (условия);
  25. В появившемся окне выберите событие ОружиеЦелевое → ЦельВышлаИзЗоныАтаки. Нажмите на кнопку Сохранить;
  26. Создайте теперь в точности такой же переход, но уже в обратную сторону(от состояния Cближение до состояние Атака) и с событием ОружиеЦелевое → ЦельВошлаВЗонуАтаки;

    Как и состояние, вы можете сделать копию перехода, нажав по очереди Ctrl+C, Ctrl+V, затем через контекстное меню (открывается по нажатию на правую кнопку мыши) можно поменять исход и цель (начало и конец).

  27. Теперь самостоятельно создайте ещё три перехода. Сперва от состояния Скан до состояния Сближение с событием Сенсор → ЦельПолучена, потом переход от состояния Бой до Скан с событием АнализаторЦели → ЦельПотеряна, затем от состояния Бой до Скан с событием АнализаторЦели → ЦельУничтожена;
  28. Для просмотра кода Вашей машины состояния нажмите правой кнопкой мыши на пустом пространстве поля редактирования, в появившемся контекстном меню выберите Посмотреть код. Код машины состояний появится в новой вкладке и автоматически перевёд Вас на неё;

    При нажатии правой кнопкой мыши на состояние или переход так же открывается контекстное меню, где вы можете посмотреть их код отдельно.

  29. Прежде чем скомпилировать схему научимся добавлять заметки;

    При нажатии правой кнопкой мыши на пустом пространстве поля редактирования, в появившемся контекстном меню выберите Добавить заметку, после чего на поле редактирования появится небольшой объект, в котором можно написать свою заметку (комментарий).

  30. На боковой панели выберите раздел Компилятор;

    Если вы не успеваете доделать схему до компиляции, то на левой панели нажмите на кнопку Меню. В появившейся вкладке нажимаем на Сохранить как... или Сохранить. Позже через меню вы можете открыть незаконченную схему.

  31. В разделе компилятора нажмите на кнопку Скомпилировать;

    Запомните! Для работы компилятора требуется подключение к интернету!

  32. Теперь экспортируем код схемы для игры «Берлога: Защита пасеки», для этого нажмите на кнопку Сохранить код в разделе Компилятор;

    Так же вы можете посмотреть код в отдельной вкладке нажав на кнопку Показать код.

    33. Машина состояний, которая в результате должна получится:

  33. После того, как мы сохранили нашу схему и удостоверились в правильности схемы, можем теперь загрузить файл в игре «Берлога: Защита пасеки»;

    В игре появится схема, которую вы сделали.

  34. Если после открытия схемы машины состояний камера редактора находится для Вас слишком близко или далеко, то справа в нижнем углу поля редактирования Вы можете приблизить или отдалить камеру. Либо зажмите клавишу Ctrl на клавиатуре и прокручивайте колёсико мыши до нужного положения камеры;
  35. Для включения/отключения отображения сетки, необходимо в правом нихнем углу поля редактирования нажать на крайнюю левую кнопку.
  36. Для сброса настроек камеры, в том числе её центрирования, нажмите правой кнопкой мыши на пустом пространстве поля редактирования, в появившемся контекстном меню выберите Центрировать камеру. После чего масштабирование сбросится до стандартного размера, а камера переместиться в центр редактора;
  37. Теперь закончим нашу работу в IDE, для этого нажмите на клавишу F11 на клавиатуре, чтобы выйти из полноэкранного режима. Затем в верхнем правом углу IDE нажмите на красную кнопку с крестиком. Если у Вас были сделанны изменения в машине состояний, то IDE потребует от Вас сделать выбор: сохранить или не сохранять последнии изменения.

Вы смогли освоиться с основным функционалом IDE, а так же создать рабочую схему для игры «Берлога: Защита пасеки»! Поздравляем!
