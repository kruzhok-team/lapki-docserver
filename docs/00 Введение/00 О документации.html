00. О документации

Разделы документации посвящены различным темам, с которыми вы столкнётесь при работе с IDE. Познакомимся с ними:

  • 00. Введение — пояснения к документации и пошаговый экскурс о работе с IDE;
  • 01. Работа с редактором — описание работы с редактором IDE;
  • 02. Работа с компилятором — описание, как работать со схемами из игры «Берлога: Защита пасеки» в IDE;
  • 03. Работа с загрузчиком — про взаимодействие Arduino и IDE;
  • 04. Дополнительные материалы — рекомендовано разработчиками для изучения таких тем, как инженерия, программирование, электроника.

Спасибо, что пользуетесь нашей IDE!
