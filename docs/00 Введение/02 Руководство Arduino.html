00. Введение

02. Руководство Arduino

Научимся пошагово работать со схемой -> с платформой Arduino в IDE.

Во время прохождения руководства, Не нажимайте комбинацию клавиш Ctrl + R, иначе клиент обновится вместе с проделанной работой!

Для лучшей работы в IDE перейдём сначала в полноэкранный режим, для этого нажмите на клавишу F11 на клавиатуре.

  1. На левой панели нажмите на кнопку Меню (значок в виде трёх полосок);
  2. В появившейся вкладке нажимаем на Создать...;
  3. В модальном окне на первой вкладке выбираем схему для Arduino Uno или Arduino Micro. После выбора схемы, появится область редактирования машин состояний;
  4. На левой панели выберите Проводник (значок в виде кубика);
  5. В появившейся вкладке нажмите кнопку Добавить... Появится модальное окно выбора компонента;
  6. Выберите компонент LED, введите название компонента и нажмите Добавить. В списке компонентов появится новый компонент с введённым ранее названием и соответствующим значком;
  7. Проделываем то же действие из пункта №6 c компонентом Timer;
  8. Дважды нажимаем левой кнопкой мыши на основном поле редактора. На месте клика появится новое пустое состояние;
  9. Дважды нажимаем левой кнопкой мыши на заголовок состояния или один раз на иконку карандаша. После переименовываем на Выключен и нажимаем на клавишу Enter;

    Для отмены переименования нажмите клавишу Esc или же нажмите левой кнопкой мыши за пределами заголовка состояния.

  10. Под заголовком дважды нажимаем левой кнопкой мыши на пустую область состояния. Появится модальное окно для выбора нового события и его действий;
  11. Вверху в левом выпадающем списке выберите событие System, а в правом выпадающем списке действие onExit.
  12. Теперь нажмите кнопку с плюсом в области Делай. В модальном окне выберите действие Timer -> Start, а в появившемся ниже блоке введите 1000. Нажмите Сохранить;

    В списке раздела Делай появится пиктографические изображения созданных Вами действий.

  13. Чтобы удалить действие из списка в разделе Делай, достаточно выбрать его и следом нажать кнопку с иконкой минуса. После чего, выбранное действие будет удалено из списка;
  14. Теперь самостоятельно создайте только что удалённое действие из пункта №12 и добавьте ещё одно действие LED1 -> off;

    Если Вы допустили ошибку в создании действия, то в списке при двойном нажатии на действие Вам откроется окно для его изменение.

  15. Зажмите левой кнопкой мыши любое Ваше созданное действие в списке раздела Делай и перенесите его поверх другого действия. После чего они поменяются местами.
  16. Создайте ещё одно состояние самостоятельно, но с такими свойствами: Имя — Включен, Событие — System -> onEnter, Действия — LED1 -> on, Timer -> Start(1000);

    Для быстрого создание подобного состояния, выберите уже созданое состояние и нажмите комбинацию клавиш Ctrl + C, а затем Ctrl + V. Поверх копируемого состояние появится его копия, нажмите на неё один раз и перенесите его вниз, чтобы оба состояния были видны в редакторе.

  17. Если Вы создали состояние с нуля, то переходите к следующему шагу, если же Вы создали копию, то переименуйте её на Включен. Затем дважды нажмите на событие и выберите System -> onEnter, потом так же измените действие LED1 -> off на LED1 -> on
  18. Нажмите правой кнопкой мыши по последнему созданному состоянию и в появившемся контекстном меню выберите раздел Редактирование., затем Назначить начальным. Рядом с выбранным состоянием появится элемент с названиемНачало, обозначающая его начальным процессом машины состояний;
  19. Наведитесь курсором на начальное состояние и нажмите на него один раз, затем переведите курсор на появившуюся на рамке состояния иконку со знаком плюс и потяните до другого состояния. Появится модальное окно для редактирования нового перехода(условия);
  20. В появившемся окне выберите событие Timer -> timeout. Нажмите на кнопку Сохранить;
  21. Создайте теперь в точности такой же переход, но уже в обратную сторону(от обычного состояния до начального);

    Как и состояние, Вы можете сделать копию перехода, нажав по очереди Ctrl + C, Ctrl + V, затем через контекстное меню поменять исход и цель (начало и конец).

  22. Для просмотра кода Вашей машины состояния нажмите правой кнопкой мыши на пустом пространстве поля редактора, в появившемся контекстном меню выберите Посмотреть код. Код машины состояний появится в новой вкладке и автоматически перевёд Вас на неё;

    При нажатии правой кнопкой мыши на состояние, переход так же открывается контекстное меню, где Вы можете посмотреть их код отдельно.

  23. Подключите плату Arduino Uno или Arduino Micro к компьютеру;
  24. На левой панели нажмите на кнопку Компилятор;

    Если Вы не успеваете доделать схему Arduino до компиляции, то на левой панели нажмите на кнопку Меню. В появившейся вкладке нажимаем на Сохранить как.... Позже через меню Вы можете открыть незаконченную схему.

  25. На левой панели нажмите на кнопку Скомпилировать;

    Запомните! Для работы компилятора требуется подключение к интернету!

  26. Теперь сохраним бинарный код схемы для Arduino, для этого нажмите на кнопку Сохранить код в разделе Компилятор;

    Так же Вы можете посмотреть код в отдельной вкладке нажав на кнопку Показать код.

  27. После того, как мы сохранили нашу схему, можем теперь прошить подключённую плату Arduino. Для этого на левой панели выберите раздел Загрузчик, выберите из списка подключенных устройств плату Arduino и ниже нажмите на кнопку Прошить. В левом нижнем углу раздела Загрузчик появится уведомление Загрузка завершена!;

    Теперь плата Arduino начнёт работать по той схеме, которую Вы сделали.

  28. Если после открытия схемы машины состояний камера редактора находится для Вас слишком близко или далеко, то справа в нижнем углу редактора Вы можете приблизить или отдалить камеру. Либо зажмите клавишу Ctrl на клавиатуре и прокручивайте колёсико мыши до нужного положения камеры;
  29. Для сброса настроек камеры, в том числе её центрирования, нажмите правой кнопкой мыши на пустом пространстве поля редактора, в появившемся контекстном меню выберите Центрировать камеру. После чего масштабирование сбросится до стандартного размера, а камера переместиться в центр редактора;
  30. Теперь закончим нашу работу в IDE, для этого нажмите на клавишу F11 на клавиатуре, чтобы выйти из полноэкранного режима. Затем в верхнем правом углу IDE нажмите на красную кнопку с крестиком. Если у Вас были сделанны изменения в машине состояний, то IDE потребует от Вас сохранить или не сохранять последнии изменения.

Вы смогли освоиться с основным функционалом IDE и создать первую рабочую схему и залить прошивку в  Arduino! Поздравляем!
