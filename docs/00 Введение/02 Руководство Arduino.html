00. Введение

02. Руководство Arduino

Научимся пошагово работать со схемой -> с платформой Arduino в IDE.

Для лучшей работы в IDE перейдём сначала в полноэкранный режим, для этого нажмите на клавишу F11 на клавиатуре.

  1. На левой панели выберите Меню (значок в виде трёх полосок);
  2. В появившейся вкладке нажимаем на Создать...;
  3. В модальном окне на первой вкладке выбираем схему для Arduino Uno или Arduino Micro. После выбора схемы, появится область редактирования машин состояний;
  4. На левой панели выберите Проводник (значок в виде кубика);
  5. В появившейся вкладке нажмите кнопку Добавить... Появится модальное окно выбора компонента;
  6. Выберите компонент LED, введите название компонента и нажмите Добавить. В списке компонентов появится новый компонент с введённым ранее названием и соответствующим значком;
  7. Проделываем то же действие из пункта №6 c компонентом Timer;
  8. Дважды кликните левой кнопкой мыши на основном поле редактора. На месте клика появится новое пустое состояние;
  9. Дважды нажимаем левой кнопкой мыши на заголовок состояния или один раз на иконку карандаша. После переименовываем на Выключен и нажимаем на клавишу Enter;
  10. Под заголовком дважды нажимаем левой кнопкой мыши на пустую область состояния. Появится модальное окно для выбора нового события и его действий;
  11. Вверху выберите в левом выпадающем списке событие System, а в правом выпадающем списке действие onExit, следом нажмите кнопку с плюсом в области Делай. В модальном окне выберите действие Timer -> Start, а в появившемся ниже блоке введите 1000. Нажмите Сохранить;
  12. Чтобы удалить из списка действие, достаточно выбрать его и следом нажать кнопку с иконкой минуса. После чего, выбранное действие будет удалено из списка;
  13. Теперь самостоятельно добавьте ещё одно действие LED1 -> off;
  14. В списке раздела Делай появится пиктографические изображения созданных Вами действий;
  15. Зажмите левой кнопкой мыши любое Ваше созданное действие в списке раздела Делай и перенесите его поверх другого действия. После чего они поменяются местами.
  16. Создайте ещё одно состояние самостоятельно, но с такими свойствами: Имя — Включен, Событие — System -> onEnter, Действия — LED1 -> on, Timer -> Start(1000);
  17. Нажмите правой кнопкой мыши по последнему созданному состоянию и в появившемся контекстном меню выберите раздел Редактирование., затем Назначить начальным. Рядом с выбранным состоянием появится нода с названиемНачало, обозначающая его начальным процессом машины состояний;
  18. Наведитесь курсором на начальное состояние и нажмите на него один раз, затем переведите курсор на появившуюся на рамке состояния иконку со знаком плюс и потяните до другого состояния. Появится модальное окно для редактирования нового перехода(условия);
  19. В появившемся окне выберите событие Timer -> timeout. Нажмите на кнопку Сохранить;
  20. Создайте теперь в точности такой же переход, но уже в обратную сторону(от обычного состояния до начального);
  21. Для просмотра кода Вашей машины состояния нажмите правой кнопкой мыши на пустом пространстве поля редактора, в появившемся контекстном меню выберите Посмотреть код. Код машины состояний появится в новой вкладке и автоматически перевёд Вас на неё;
  22. Теперь сохраним созданную машину состояний, для этого на левой панели выберите Меню (значок в виде трёх полосок);
  23. В появившейся вкладке нажимаем на Сохранить как...;
  24. Выберите папку, куда Вы хотите сохранить свою машину состояний в формате .graphml;
  25. Теперь попробуем открыть Вашу сохранённую машину состояний, для этого на левой панели выберите раздел Меню (значок в виде трёх полосок);
  26. В появившейся вкладке нажимаем на Открыть... и выбираем сохранённую Вами ранее схему машины состояний;
  27. В области редактора появится Ваша схема машины состояний.
  28. Если после открытия схемы машины состояний камера редактора находится для Вас слишком близко или далеко, то справа в нижнем углу редактора Вы можете приблизить или отдалить камеру. Либо зажмите клавишу Ctrl на клавиатуре и прокручивайте колёсико мыши до нужного положения камеры;
  29. Для сброса настроек камеры, в том числе её центрирования, нажмите правой кнопкой мыши на пустом пространстве поля редактора, в появившемся контекстном меню выберите Центрировать камеру. После чего масштабирование сбросится до стандартного размера, а камера переместиться в центр редактора;
  30. Теперь закончим нашу работу в IDE, для этого нажмите на клавишу F11 на клавиатуре, чтобы выйти из полноэкранного режима. После в верхнем правом углу IDE нажмите на красную кнопку с крестиком. Если у Вас были сделанны изменения в машине состояний, то приложение потребует от Вас сохранить или не сохранять последнии изменения.

Вы смогли освоиться с основным функционалом IDE и создать первую машину состояний для Arduino! Поздравляем!
