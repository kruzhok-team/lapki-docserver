02. Быстрый старт

02. Машина состояний

После создания документа появится графическое поле редактирования машины состояний (холст), а сбоку откроется раздел «Диаграмма» .

КиберМишка

МС-ТЮК

Холст можно перемещать с помощью зажатой правой кнопки мыши, масштабировать с помощью зажатой кнопки Ctrl и колёсика мыши, либо с помощью кнопок в правом нижнем углу ( для приближения и для отдаления). Нажав на значок сетки в правом нижнем углу, можно переключить отображение сетки.

В боковом разделе «Диаграмма» в списке «Машины состояний» будет показана, созданная машина состояний с автоматически сгенерированным именем (CyBear1, либо MS1Main1). Давайте дадим ей осмысленное имя. Для этого дважды кликните по ней, откроется окно в котором можно задать ей имя. Назовите её «Мигалка».

Также через это окно можно удалить машину состояний или создать её дубликат, но это не пригодится в рамках текущей задачи.

КиберМишка

МС-ТЮК

Чтобы создать новую машину состояний нажмите на значок плюса , который находится напротив заголовка «Машины состояний» (это не пригодится в рамках текущей задачи).

Чтобы открыть вкладку со схемой машины состояния, дважды кликните по ней в списке на вкладке «Диаграмма».
