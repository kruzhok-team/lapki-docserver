02. Быстрый старт

01. Создание документа

Для более удобной работы можно перейти в режим полного экрана, нажав на клавишу F11. Для выхода из полноэкранного режима нужно нажать F11 повторно.

Создадим новый документ. На боковой панели при запуске IDE будет открыт раздел Документ. Если его нет, нажмите на значок документа в левом верхнем углу. Нажмите «Создать...». Откроется диалоговое окно создания документа.

КиберМишка

Найдите платформу «КиберМишка» из правого списка и кликните два раза по ней левой кнопкой мыши, либо перетащите её, зажимая левую кнопку мыши пока она не окажется в правом столбце.


МС-ТЮК

Выберите платформу «МС-ТЮК-1: Центральная плата» из правого списка и кликните два раза по ней левой кнопкой мыши, либо перетащите её, зажимая левую кнопку мыши пока она не окажется в правом столбце.

После того, как Вы добавите платформу, она должна появиться в левом списке. нажмите кнопку «Создать» в правом нижнем углу.
