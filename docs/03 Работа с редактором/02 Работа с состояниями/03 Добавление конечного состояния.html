03. Добавление конечного состояния

Нажмите правой кнопкой мыши в поле редактирования и в появившемся контекстном меню выберите «Вставить конечное состояние». В поле редактирования появится окружность с кольцом.

Конечное состояние необходимо для явного указания завершения процесса, а также позволяет завершать работу какого-либо состояния через переход.
