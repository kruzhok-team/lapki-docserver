02. Сохранение файла

Чтобы сохранить файл в новое место на компьютере, нажмите на «Документ» на боковой панели, в открывшейся вкладке выберите «Сохранить как...». Откроется окно приложения с выбором названия и пути сохранения файла. При выборе по заданному пути появится файл с заданным названием и расширением .graphml.

Чтобы сохранить уже давно созданный файл, нажмите на «Документ» на боковой панели, в открывшейся вкладке выберите «Сохранить». Файл сохранится по тому же пути.

Для быстрого сохранения, нажмите сочетание клавиш «Ctrl+S». Для быстрого сохранения как..., нажмите сочетание клавиш «Ctrl+Shift + A».
