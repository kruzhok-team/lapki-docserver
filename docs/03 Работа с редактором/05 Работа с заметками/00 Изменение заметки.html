00. Изменение заметки

Кликните дважды левой кнопкой мыши на заметку, после чего фокус будет переведён на внутренний блок заметки, в котором можно написать желаемый Вами текст.

Для сохранения изменений нажмите за пределами области заметки, а для отмены действия редактирования нажмите на клавишу «Esc».
