07. Вставка копии условного перехода

Нажмите правой кнопкой мыши на поле редактирования, в появившемся контекстном меню выберите «Вставить». В поле редактирования появится копия условного перехода.

Для быстрой вставки копии, нажмите левой кнопкой мыши на  редактор, затем нажмите сочетание клавиш «Ctrl+V».
