03. Интерфейс среды

05. Руководство пользователя

Кнопка руководства пользователя на боковой панели открывает, либо закрывает руководство, которое Вы сейчас читаете.

В области руководства находится два раздела:

  • Содержание — включает в себя дерево руководства в котором при нажатии на какую-либо страницу, справа  появится раздел «Просмотр»;
  • Просмотр — раздел для просмотра открытой страницы и позволяющий переключаться между ними c помощью клавиш «Назад» и «Вперёд».

При наведении на левый край области руководства у Вас изменится указатель мыши, после чего, Вы можете зажать левую кнопку мыши и сдвинуть её влево (вправо), чтобы растянуть (сжать) область руководства по Вашему усмотрению.

Также открыть (закрыть) область руководства можно при нажатии клавиши «F1» на клавиатуре.
