Вкладка Компоненты позволяет Вам просмотреть список имеющихся компонентов, а также следующие действия с ними:

  1. Добавление компонента. Чтобы добавить новый компонент, нажмите на значок в виде плюса , который находится напротив заголовка «Компоненты». Появится диалоговое окно выбора компонента. Выберите компонент из списка нажмите «Добавить». Откроется окно редактирования компонента, укажите нужные Вам параметры и затем нажмите кнопку «Применить».
  2. Редактирование компонента. Для того, чтобы открыть окно редактирования компонента, дважды нажмите по нему левой кнопкой мыши, либо нажмите правой кнопкой мыши по нему.
  3. Удаление компонента. Для того, чтобы удалить компонент, откройте его окно редактирования и нажмите соответствующую кнопку.

    Удаление компонента можно отменить, нажав комбинацию клавиш Ctrl + Z, либо нажав на значок отмены действий, который находится в правом нижнем углу видимой части холста, либо зайдя в боковую вкладку История изменений, нажав там кнопку «Назад».
