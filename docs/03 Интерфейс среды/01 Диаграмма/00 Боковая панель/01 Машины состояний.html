01. Машины состояний

Вкладка Машины состояний позволяет Вам просмотреть список текущих машин состояний, а также следующие действия с ними:

  1. Добавление машины состояний. Чтобы создать новую машину состояний, нажмите на значок в виде плюса , который находится напротив заголовка «Машины состояний». Откроется окно создания машины состояния. Вам необходимо указать для неё платформу. По желанию можно задать ей имя, если этого не сделать, то будет использоваться автоматически сгенерированное имя.
  2. Редактирование машины состояний. Для того, чтобы открыть окно редактирования машины состояний, нажмите правой кнопкой мыши кнопкой по ней в списке машин состояний.
  3. Переименование машины состояний. Для того, чтобы переименовывать машину состояний, откройте её окно редактирования и введите новое имя, после чего нажмите на кнопку «Применить».
  4. Удаление машины состояний. Для того, чтобы удалить машину состояний, откройте её окно редактирования и нажмите соответствующую кнопку.

    Удаление машины состояний нельзя отменить!

  5. Дублирование машины состояний. Для того, чтобы создать копию машины состояний, откройте её окно редактирования и нажмите кнопку «Дублировать».
