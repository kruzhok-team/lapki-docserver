05. Дополнительные операции

Помимо загрузок прошивок, загрузчик предоставляет другие операции с отмеченными устройствами из таблицы.

  1. Пинг — отправить пинг на выбранные устройства, в журнале действий придут сообщения о том, ответило ли устройство на пинг или нет;
  2. Перезагрузка — перезагрузить выбранные устройства;
  3. Получить метаданные — узнать тип платы и другую информацию (метаданные), с которой можно ознакомиться через окно адресной книги (эта функция доступна только для МС-ТЮК);
  4. Выгрузка прошивки — при нажатии на эту кнопку, откроется диалоговое окно выбора места для папки, куда будут загружены прошивки, содерщащиеся на выбранных устройствах, после чего начнётся выгрузка.

Эти операции могут быть недоступны для некоторых типов плат, в таком случае, в журнале действий выведется ошибка.
