Помимо загрузок прошивок, загрузчик предоставляет другие операции с отмеченными устройствами из таблицы. По-умолчанию, они скрыты, чтобы открыть к ним доступ, необходимо переключить загрузчик в продвинутый режим, для этого нужно нажать на кнопку Продвинутый. После этого рядом с кнопкой прошивки, появятся другие кнопки для продвинутых операций.

  1. Прошить с проверкой — Загрузить прошивку с проверкой целостности. Увеличивает общее время загрузки;
  2. Пинг — Окликнуть отмеченные платы, чтобы проверить связь с ней;
  3. Сброс — перезагрузить выбранные устройства;
  4. Метаданные — узнать тип платы и другую информацию (метаданные), с которой можно ознакомиться через окно адресной книги;
  5. Скачать прошивку — Выгрузить файлы прошивки из выбранных плат.
  6. Загрузить «заводскую» прошивку — Загрузить в выбранные платы прошивку, поставляемую с новой платой. Обычно это демонстрационная прошивка;

Эти операции могут быть недоступны для некоторых типов плат, в таком случае, в журнале действий выведется ошибка.
