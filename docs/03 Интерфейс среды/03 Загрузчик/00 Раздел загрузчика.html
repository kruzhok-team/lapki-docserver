00. Раздел загрузчика

Модуль загрузчика предназначен для работы с подключёнными устройствами, в том числе для загрузки скомпилированных машин состояний на платы.

Обратите внимание, что при открытии загрузчика открывается вкладка не сбоку, а по центру.

  1. Подключение платы — выбор устройств для работы с загрузчиком;
  2. Адреса плат МС-ТЮК  — обзор таблицы с платами МС-ТЮК;
  3. Таблица с устройствами — обзор таблицы;
  4. Загрузка прошивок — как прошить устройства;
  5. Дополнительные операции — обзор дополнительных операций с устройствами;
  6. Журнал действий — обзор текстового поля внизу;
  7. Монитор порта — принятие или отправка различных данных на последовательный порт;
  8. Переключение между локальным и удалённым загрузчиком — смена адреса загрузчика;
  9. Ошибки подключения к загрузчику — что делать, если что-то пошло не так;
  10. Установка AVRDude — как установить сторонее приложение для прошивки устройств Arduino;
