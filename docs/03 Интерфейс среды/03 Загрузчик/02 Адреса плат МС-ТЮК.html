02. Адреса плат МС-ТЮК

Особенность МС-ТЮК является то, что все платы имеют уникальный адрес, и для того, чтобы взаимдойствовать с конкретной платой (например, для прошивки) необходимо указать её адрес.

Для удобства работы с МС-ТЮК, IDE хранит адреса плат в адресной книге, которая открывается по нажатию на кнопку Адреса плат МС-ТЮК.

Адресная книга — это таблица, состоящая из трёх столбцов:

  1. Название: наименование устройства, которое задаётся пользователем, оно хранится только в IDE, не оказывая никакого влияния на устройство;
  2. Адрес: идентификатор устройства;
  3. Тип: вид устройства

При двойном клике на какую-либо запись открывается его окно редактирования, где можно поменять название.

Для того, чтобы добавить новую запись нужно нажать на кнопку +. Откроется окно добавления записи. После заполнения полей, нажмите на кнопку Добавить, чтобы сохранить запись. После этого её можно будет загрузить при следующем запуске IDE.

Название устройства - это не обязательное для заполнения поле, оно служит для удобства, чтобы было легче найти устройство.

Для того, чтобы удалить запись кликните на одну из записей в книге и затем нажмите на кнопку -.

Удаление записи нельзя отменить!

Благодаря этой таблице можно сразу же добавить нужные Вам для работы платы МС-ТЮК в таблицу устройств на вкладке загрузчика без перевода в режим адресации.
