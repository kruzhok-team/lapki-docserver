03. Результат работы компилятора

Для того, чтобы увидеть результаты компиляции конкретной машины состояния, необходимо выбрать её из выпадающего меню, которое находится внизу вместе с другими кнопками. В зависимости от типа платформы машины состояния, будут доступны разные кнопки.

  • Берлога

    • При успешной компиляции внизу будет доступно только две кнопки:

    «Показать код» — В верхней панели IDE откроется вкладка с кодом схемы для игры «Берлога: Защита пасеки».

    «Сохранить код» — Открывается окно, в котором необходимо выбрать место на компьютере, в которую будет сохранена загружаемая схема для игры «Берлога: Защита пасеки».

    Недоступные кнопки необходимы лишь для схем других платформ, таких как Arduino.

    Наведите курсор мыши на раздел компилятора и прокрутите вниз колесиком мыши, чтобы увидеть остальные кнопки.

  • Arduino и МС-ТЮК

    • При успешной компиляции внизу раздела «Компилятор» станут доступны кнопки:

    «Показать журнал компиляции» — Открывается вкладка с выводом (stdout и stderr) от компилятора. Если во время компиляции произошли ошибки (NOTOK), то после stderr будет находиться текст с ошибками вашей программы, если же компиляция прошла успешно (OK), после stderr не будет текста .

    «Сохранить результат» — При нажатии на кнопку открывается окно приложения для сохранения бинарного кода в файл. Если вы скомпилировали схему, то вы можете сразу загрузить ее в плату Arduino через раздел Загрузчик, но при открытии другой схемы или закрытии IDE, схему нужно будет компилировать заново. Если же вы сохраните бинарные файлы, вы сможете потом загружать их в плату в любой момент через Загрузчик без компиляции, то есть даже при отсутствии интернета.

    «Сохранить код» — Открывается окно прлиожения, в котором необходимо выбрать папку на компьютере, в которую будет сохранен код для Arduino.

    «Показать код» — В верхней панели IDE откроются вкладки с кодом для Arduino, созданном на основе вашей схемы.

    «Прошить...» — Открывается вкладка Загрузчик.
