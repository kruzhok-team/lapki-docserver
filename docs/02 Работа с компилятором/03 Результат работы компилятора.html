02. Работа с компилятором

03. Результат работы компилятора

  • «Берлога: Защита пасеки»

    • При успешной компиляции внизу будет доступно только две кнопки:

    «Показать код» — В верхней панели IDE откроется вкладка с кодом схемы для игры «Берлога: Защита пасеки».

    «Сохранить код» — Открывается окно, в котором необходимо выбрать место на компьютере, в которую будет сохранена загружаемая схема для игры «Берлога: Защита пасеки».

    Недоступные кнопки необходимы лишь для схем Arduino.

    Наведите курсор мыши на раздел компилятора и прокрутите вниз колесиком мыши, чтобы увидеть остальные кнопки.

  • Arduino

    • При успешной компиляции внизу раздела компилятор станут доступны кнопки:

    «Показать stderr» — Если во время компиляции произошли ошибки (NOTOK), то при нажатии на кнопку откроется вкладка с ошибками вашей программы, если же компиляция прошла успешно (OK), то откроется пустая вкладка.

    «Показать stdout» — Открывается вкладка со справочной информацией.

    «Сохранить результат» — При нажатии на кнопку открывается окно для сохранения бинарного кода в файл. Если вы скомпилировали схему,то вы можете сразу загрузить ее в плату Arduino через раздел Загрузчик, но при открытии другой схемы или закрытии IDE, схему нужно будет компилировать заново. Если же вы сохраните бинарные файлы, вы сможете потом загружать их в плату в любой момент через Загрузчик без компиляции, то есть даже при отсутствии интернета.

    «Показать код» — В верхней панели IDE откроются вкладки с кодом для Arduino, созданном на основе вашей схемы.

    «Сохранить код» — Открывается окно, в котором необходимо выбрать папку на компьютере, в которую будет сохранен код для Arduino.

    «Прошить код» — Открывается вкладка Загрузчик.
