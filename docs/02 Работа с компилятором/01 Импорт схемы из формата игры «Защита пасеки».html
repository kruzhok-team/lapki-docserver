02. Работа с компилятором

01. Импорт схемы из формата игры «Берлога: Защита пасеки»

Для действия потребуется подключение к интернету!

Откройте Главное меню на левой панели, выберите пункт Импорт . В проводнике выберите файл схемы из игры «Берлога: Защита пасеки» и нажмите OK. При успешном чтении и преобразовании в основном поле редактора появится схема, соответствующая записанной в файл.

Для быстрого импорта файла, нажмите комбинацию клавиш Ctrl+I.
