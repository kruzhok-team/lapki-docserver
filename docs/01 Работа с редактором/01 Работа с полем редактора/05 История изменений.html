05. История изменений

Чтобы посмотреть выполненные изменения в редакторе, нажмите на Историю изменений на левой панели, в открывшейся вкладке просматривайте, отменяйте, возвращайте последнии изменения, которые Вы совершили в редакторе.

Для быстрой отмены последнего действия, нажмите комбинацию клавиш Ctrl + Z. Для быстрого возврата последнего действия, нажмите комбинацию клавиш Ctrl + Y.
