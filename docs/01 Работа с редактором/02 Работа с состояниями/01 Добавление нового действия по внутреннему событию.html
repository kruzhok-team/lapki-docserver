01. Добавление нового действия по внутреннему событию

Дважды кликните левой кнопкой мыши по существующему состоянию (при его отсутствии, создайте новое). Появится модальное окно для выбора нового события.

Выберите необходимое событие, следом нажмите кнопку с плюсом в области Делай. В появившемся окне выбора действия выберите действие, нажмите Сохранить. В отредактированном ребре появится пиктографическое изображение выбранного действия.
