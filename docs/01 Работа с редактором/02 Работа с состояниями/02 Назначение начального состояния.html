02. Назначение начального состояния

Нажмите правой кнопкой мыши по состоянию, в появившемся контекстном меню выберите раздел Редактирование., затем Назначить начальным. Рядом с выбранным состоянием появится нода с названиемНачало, обозначающая его начальным.
