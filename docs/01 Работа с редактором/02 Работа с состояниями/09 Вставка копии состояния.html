09. Вставка копии состояния

Нажмите правой кнопкой мыши по области редактора, в появившемся контекстном меню выберите Вставить. В редакторе появится копия состояния.

Для быстрой вставки копии, нажмите левой кнопкой мыши на  редактор, затем нажмите комбинацию клавиш Ctrl + V.
