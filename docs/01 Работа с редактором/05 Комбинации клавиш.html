01. Работа с редактором

05. Комбинации клавиш

  1. Ctrl + N — Cоздать новый файл в формате .graphml;
  2. Ctrl + O — Открыть файл в формате .graphml;
  3. Ctrl + I — Импортировать файл из игры Берлога: «Защита пасеки» в формате .graphml;
  4. Ctrl + S — Сохранить файл в формате .graphml;
  5. Ctrl + Shift + A — Сохранить новый файл или уже созданный в формате .graphml в любое место на компьютере;
  6. Ctrl + C — Копировать выбранное состояние или связь;
  7. Ctrl + V — Вставить скопированное состояние или связь. Элементы копируются в точности с оригиналом, поэтому они будут находится на тех же координатах и иметь те же свойства;
  8. Ctrl + Delete(Del) — Удалить состояние, связь, событие, действие, компонент;
  9. Ctrl + Y — Вернуть последнее выполненное действие в редакторе;
  10. Ctrl + Z — Отменить последнее выполненное действие в редакторе;
  11. Ctrl + R — Обновление IDE;
  12. F11 — Полноэкранный режим;
  13. F12 — Показать/скрыть консоль отладки (в режиме разработчика).
