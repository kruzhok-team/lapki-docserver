01. Работа с редактором

05. Комбинации клавиш

1. Ctrl + N — создать новый файл в формате .graphml;

2. Ctrl + O — Открыть файл в формате .graphml;

3. Ctrl + S — Сохранить файл в формате .graphml;

4. Ctrl + Shift + A — Сохранить новый файл или уже созданный в формате .graphml в любое место на компьютере;

5. Ctrl + C — Копировать выбранное состояние(ноду) или связь;

6. Ctrl + V — Вставить скопированное состояние(ноду) или связь. Элементы копируются в точности с оригиналом, поэтому они будут находится на тех же координатах и иметь те же свойства;

7. Ctrl + Delete(Del) — Удаление состояния, связи, события, действия;

8. Ctrl + Y — Вернуть последнее выполненное действие;

9. Ctrl + Z — Отменить последнее выполненное действие;

10. F11 — Полноэкранный режим;

11. F12 — Показ(Скрытие) DevTools;
