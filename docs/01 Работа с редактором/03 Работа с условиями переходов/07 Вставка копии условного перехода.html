07. Вставка копии условного перехода

Нажмите правой кнопкой мыши по области редактора, в появившемся контекстном меню выберите Вставить. В редакторе появится копия условного перехода.

Для быстрой вставки копии, нажмите левой кнопкой мыши на  редактор, затем нажмите комбинацию клавиш Ctrl + V.
