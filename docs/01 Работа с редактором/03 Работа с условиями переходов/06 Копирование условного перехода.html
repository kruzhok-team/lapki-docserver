06. Копирование условного перехода

Нажмите правой кнопкой мыши по условному переходу, в появившемся контекстном меню выберите Копировать. Код выбранного условного перехода скопируется в буфер.

Для быстрого копирования, нажмите левой кнопкой мыши на  условный переход, затем нажмите комбинацию клавиш Ctrl + C.
