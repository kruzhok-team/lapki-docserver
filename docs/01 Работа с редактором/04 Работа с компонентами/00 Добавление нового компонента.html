01. Работа с редактором 04. Работа с компонентами

00. Добавление нового компонента

На левой панели выберите Проводник (значок в виде кубика), в появившейся вкладке нажмите кнопку Добавить... Появится модальное окно выбора компонента.

Выберите компонент для Arduino, введите название компонента и нажмите Добавить. В списке компонентов появится новый компонент с введённым ранее названием и соответствующим значком.

Выберите компонент для Берлога: Защита пасеки. В списке компонентов появится новый компонент с  названием и соответствующим значком.
