01. Работа с редактором

11. Добавление внешнего перехода с действием

Дважды кликните левой кнопкой мыши на существующем внешнем переходе (при его отсутствии, создайте новый внешний переход). Появится модальное окно для редактирования перехода.

Нажмите кнопку с плюсом в области Действия. В появившемся окне выбора действия выберите действие, нажмите Сохранить, затем снова Сохранить. В отредактированном ребре появится пиктографическое изображение выбранного действия.
