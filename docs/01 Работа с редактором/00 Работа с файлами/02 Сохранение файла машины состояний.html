02. Сохранение файла машины состояний

Чтобы сохранить новый проект или сохранить уже старый файл в новое место на компьютере, нажмите на Главное меню на левой панели, в открывшейся вкладке выберите Сохранить как.... Откроется меню с выбором названия и пути сохранения файла. При выборе по заданному пути появится файл с заданным названием и расширением .graphml.

Чтобы сохранить открытый проект, нажмите на Главное меню на левой панели, в открывшейся вкладке выберите Сохранить. Файл сохранится по тому же пути.

Для быстрого сохранения, нажмите комбинацию клавиш Ctrl + S. Для быстрого сохранения как..., нажмите комбинацию клавиш Ctrl + Shift + A.
