01. Работа с редактором → 00. Работа с файлами

00. Создание нового файла машины состояний

Нажмите на Главное меню на левой панели, в открывшейся вкладке выберите Создать, после выберите целевую платформу. На верхней панели появится вкладка редактора, также появится пустое основное поле редактора схемы.

Для быстрого создание файла, нажмите комбинацию клавиш Ctrl + N.
