03. Работа с Arduino

05. Загрузка прошивки в плату Arduino Uno

Откройте схему для платформы Arduino. Подключите к компьютеру Arduino Uno или подключитесь к удалённому загрузчику. Отобразите подключенные устройства, выберите нужное устройство в списке и нажмите кнопку Прошить.

Начнётся загрузка прошивки, после её завершения плата Arduino начнёт выполнять программу, соответствующую открытой схеме.
