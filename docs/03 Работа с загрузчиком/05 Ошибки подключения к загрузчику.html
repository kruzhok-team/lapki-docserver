03. Работа с загрузчиком

05. Ошибки подключение к загрузчику

При возникновении ошибок подключения к загрузчику, в разделе загрузчика появится кнопка Подробнее. При нажатии на эту кнопку откроется окно с объяснением ошибки.

При отсутствии соединения с локальным (удалённым) загрузчиком, кнопка Обновить заменяется на кнопку Перезапустить (Переподключиться). При нажатии начнётся перезагрузка (переподключение) локального (удалённого) загрузчика.

При неудачном соединении или потере его с удалённым загрузчиком IDE попытается несколько раз автоматически переподключиться (исключение: если возникла ошибка, связанная с некорректными настройками вебсокетов).
