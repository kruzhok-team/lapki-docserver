03. Работа с загрузчиком

04. Загрузка прошивки в плату Arduino

Откройте файл с прошивкой для Arduino (с расширением .hex) через кнопку ... или создайте и скомпилируйте свою через IDE. Подключите к компьютеру плату Arduino или подключитесь к удалённому загрузчику. Выберите нужное устройство в списке и нажмите кнопку Загрузить.

Если Вы открыли не тот файл, который Вам нужен была для прошивки, то нажмите на кнопку с иконкой X и выберите другую подходящую схему.

Если под кнопкой Загрузить, появилась надпись типа из файла file.hex, где file - это название файла, значит, этот файл будет выбран для загрузки в устройство. Если эта надпись отсутствует, для прошивки будет использоваться результат последней успешной компиляции схемы Arduino через IDE.

Начнётся загрузка прошивки, после её завершения плата Arduino начнёт выполнять программу, соответствующей загруженной схеме, а в левом нижнем углу раздела Загрузчик появится уведомление Загрузка завершена!, если прошивка не удалась, то в том же месте появится уведомление об ошибке.

Помните, что при работе с удалённым загрузчиком нельзя выбрать устройство, которое в данный момент прошивается другим клиентом. Если вы выбрали такое устройство, то загрузчик отклонит Ваш запрос.
