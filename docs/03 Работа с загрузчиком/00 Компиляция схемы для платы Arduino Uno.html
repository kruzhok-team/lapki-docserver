03. Работа с Arduino

00. Компиляция схемы для платы Arduino Uno

При открытом файле схемы для платформы Arduino перейдите на вкладку Компилятор на левой панели и нажмите кнопку Скомпилировать. Запустится преобразование схемы, по завершении которого на этой же вкладке появится появится информация о результате компиляции.

Далее можно просмотреть промежуточный код, сохранить файл прошивки или загрузить прошивку в устройство.
