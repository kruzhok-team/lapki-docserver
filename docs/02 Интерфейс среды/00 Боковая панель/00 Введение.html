02. Интерфейс среды -> 00. Боковая панель

00. Введение

Слева находится боковая панель для взаимодействия с IDE. Изучим каждый раздел боковой панели:

  1. Документ — включает в себя создание проекта, открытие файлов, импорт схемы из игры «Берлога: Защита пасеки», сохранение текущего проекта в файл, а также готовые примеры проектов для ознакомления;
  2. Диаграмма — здесь находятся список машин состояний, задейственных в текущем проекте, список компонентов и иерархия состояний;
  3. Компилятор — преобразование схемы машины состояний в формат выбранной целевой платформы;
  4. Загрузчик — загрузка скомпилированной схемы машины состояний в подключенное устройство,  работа с монитором порта и другие действия с устройствами.
  5. История изменений — возврат или отмена последних действий в редакторе IDE, а также их краткое описание;
  6. Документация — открывает и закрывает документацию по нажатию на иконку.
  7. Настройки — содержит настраиваемые элементы IDE, например, тема приложения (светлая и тёмная), выбор адреса для компилятора и т.д.
